# Bug Report - Saucedemo

# BR002

**Cenário:** Após logar com o perfil "problem_user", foi notado que todas as fotos dos produtos foram trocadas pela de um cachorro segurando uma bola de tênis

**Perfis Afetados:** problem_user

**Severidade:** Média (O bug apesar de não impedir diretamente o usuário de realizar uma compra, inibe a vontade de comprar ou sequer se aprofundar no produto, pois a imagem neste caso serve não somente como parte estética mas tambem tem parte funcional para entender melhor o produto)

**Prioridade:** Alta (considerando o site como projeto já em produção, a ausência de imagens reais dos produtos compromete diretamente o interesse de compra dos clientes, afetando todo o catálogo)

**Passos:**
1. Acessar https://www.saucedemo.com/
2. Logar com o perfil "problem_user"
3. Visualizar as imagens na tela inicial dos itens do site.

**Resultado esperado:** Após logar com o perfil "problem_user" as imagens dos produtos devem ser as características de cada produto

**Resultado obtido:** Após logar com o perfil "problem_user" é averiguado que as imagens de cada produto foram trocadas pela foto de um cachorro segurando uma bola de tênis
